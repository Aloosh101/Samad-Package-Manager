Thank you for your interest in contributing to SPM! 

To protect the project and ensure we can maintain, dual-license, or commercialize the project in the future, all contributors must agree to the Contributor License Agreement (CLA) below before their contributions can be merged.

## How to Contribute

1. Fork the repository.
2. Create a new branch for your feature or bug fix.
3. Write clear, documented code with tests if applicable.
4. Ensure your commits are clean.
5. Open a Pull Request and state in your description: *"I agree to the Contributor License Agreement in CONTRIBUTING.md."*
